# Hunt TMS Carrier Side MVP

Carrier-side MVP interface for Hunt TMS.

## Stack

- React
- Vite
- Tailwind CSS
- lucide-react
- Recharts

## Modules

- Home / Operations Command Center
- Equipment
- Drivers
- Timeline
- Loads
- Map
- Invoices
- Expenses
- Salaries
- Reports
- Settings

## Local run

npm install
npm run dev

## Production build

npm run build
